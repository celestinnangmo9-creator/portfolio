PORTFOLIO — NANGMO NGUEMO CELESTIN ANGE
=========================================

CONTENU DU DOSSIER
-------------------
index.html              -> la page du portfolio
assets/css/style.css    -> tous les styles
assets/js/main.js       -> menu mobile, bouton FR/EN, animations
assets/img/celestin.jpg -> ta photo
assets/docs/CV_...pdf   -> ton CV (bouton "Télécharger le CV")

COMMENT L'OUVRIR
-----------------
Double-clique sur index.html : il s'ouvre dans ton navigateur.
Pour le mettre en ligne gratuitement : dépose tout le dossier sur
Netlify Drop (netlify.com/drop) ou GitHub Pages.

À FAIRE AVANT DE PUBLIER
-------------------------
1. Dans index.html, cherche la ligne :
      <a class="contact__card" href="https://github.com/" ...>
   et remplace "https://github.com/" par ton vrai lien GitHub
   (et le texte "Lien à venir" par ton pseudo si tu veux).

2. Si tu as un profil LinkedIn, dis-le-moi et je l'ajoute.

3. Si tu veux ajouter des captures d'écran ou des liens en ligne
   pour tes 3 projets (soutenance, GazLivré, Manger Facilement
   Chez Vous), envoie-les-moi et je les intègre aux fiches projet.

BILINGUE
--------
Le bouton FR / EN en haut à droite bascule tout le texte du site
sans recharger la page. Les traductions sont dans assets/js/main.js
si tu veux les ajuster toi-même plus tard.
